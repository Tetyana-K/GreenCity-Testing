# GreenCity Tests

## Короткий опис завдання
Мета цього репозиторію — створення тестів для сторінки подій GreenCity.  
Виконано аналіз функціональності сторінки та створено 3 позитивні тест-кейси  та 1 негативний у форматі Markdown (.md) 

## Посилання на сторінку, що тестується
[GreenCity Events](https://www.greencity.cx.ua/#/greenCity/events)

## Автор роботи
Тетяна Кирик

## Завдання
1. Створити власний репозиторій на GitHub:
   - Назва репозиторію: `greencity-tests` (або інша зрозуміла назва)
   - Репозиторій має бути публічним

2. Проаналізувати сторінку новин GreenCity:
   - Перейдіть за посиланням: [https://www.greencity.cx.ua/#/greenCity/events](https://www.greencity.cx.ua/#/greenCity/events)
   - Ознайомтесь із функціональністю сторінки

3. Створити 3 тест-кейси для перевірки функціональності сторінки.

4. Описати тест-кейси у форматі `.md`  
   Кожен тест-кейс повинен містити:
   - **Title** — назва тесту
   - **Preconditions** — передумови
   - **Test Steps**, оформлені у вигляді таблиці:

| Step | Action | Data | Expected Result |
|------|--------|------|----------------|
| 1    | ...    | ...  | ...            |
| 2    | ...    | ...  | ...            |
